## Features
- Romantic animated UI with floating hearts
- Interactive "Will you be my Valentine?" question
- Fun "No" button behavior (it shrinks while "Yes" grows!)
- Confetti celebration on "Yes"
- Hidden love letter reveal
- Simple Express backend

## Run Locally

```bash
npm install
npm start
```

Visit `http://localhost:3000`

## Customize

Edit `public/index.html`:
- Change "My Dearest" to their name
- Modify the messages and love letter
- Adjust colors in the `:root` CSS variables
